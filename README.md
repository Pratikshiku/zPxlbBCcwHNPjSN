## Graduation Thesis Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/4bf7b559788d492984c18a7985c6634d/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users, supervising teachers, reviewing teachers, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Topic Information
6. Topic Application
7. Topic Tasks
8. Thesis
9. Announcement Information
10. Reviewing Teacher Information
11. User Information
12. Supervising Teacher Information



**Supervising Teacher Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Topic Information Management
6. Topic Application Management
7. Topic Task Management
8. Thesis Management
9. Announcement Information Management
10. User Management



**Reviewing Teacher Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Thesis Management
6. Announcement Information Management
7. User Management




**Administrator Functions:**
1. Login
2. Personal Center (change password, personal information)
3. Administrator Management
4. Thesis Management
5. Topic Information Management
6. Topic Application Management
7. Topic Task Management
8. Basic Data Management (class management, topic type management, task type management, announcement type management, college management)
9. Announcement Information Management
10. Reviewing Teacher Management
11. User Management
12. Supervising Teacher Management
